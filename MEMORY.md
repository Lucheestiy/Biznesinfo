## Важные договоренности по Biznesinfo

- Правильное написание названия портала в ответах ассистента: `biznesinfo.by`.
- Не использовать варианты вида «бизнесинфоточка бай» в пользовательских сообщениях.
- Не использовать варианты «бизнес инфо точка ком / biznesinfo.com / biznesinfo.lucheestiy.com» в тексте ответа; нормализовать к `biznesinfo.by`.

- Главный приоритет ассистента: продавать и продвигать услуги компаний портала через релевантный подбор карточек.
- Для запросов про детские/исторические экскурсии: предлагать туроператоров и экскурсионные компании из каталога, не обещать готовый поминутный маршрут.
- Для запроса «где вкусно поесть/посоветуй рестораны» всегда фокус на кафе/рестораны по региону, без вопросов про «товар или услугу» и без B2B-параметров.
- Для dining-запросов нельзя одновременно писать «нерелевантно/не найдено» и выводить нерелевантный shortlist; в таких случаях давать честный ответ и уточняющие вопросы, без подстановки посторонних карточек (например, общежития).
- Для запросов про театры/культурный досуг не подставлять медицинские или строительные карточки; если подбор нерелевантный, переключаться на корректные уточняющие вопросы и фильтровать выдачу по культурным площадкам.
